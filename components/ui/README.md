# Patterns

## Categorization

Categories help everyone find components easily in the design kit or style guide. Categories are mainly used on the style guide for break down but not used in the design kit as it makes items harder to find.

- Foundation
  - Grid
  - Layout
  - Structurep
  - Navigation
- Controls
  - Buttons
  - Links
  - Input fields
  - Checkboxes
  - Radios
  - Toggles
  - Tabs
- Overlays
  - Modals
  - Popovers
  - Tooltips
- Feedback
  - Banners
  - Alerts
  - Toasts
  - Loaders
- Text
  - Titles/Headings
  - Body text
- Data Visualization
  - Charts / Graphs
- Content
  - Chips
  - Tags
  - Cards
